
# primitive type /reference type

## Primitive Type

```java
byte short int long float double boolean char
```
- 定义基本类型，让Java在内存中某比特段之间存放变量的数据（创建了一个盒子，将数据放入），但不可知盒子的位置（与C语言不同）

> **The Golden Rule of Equals**
> `When you write `y = x`, you are telling the Java interpreter to copy the bits from x into y.`

## Reference Types

当我们声明任意reference type时，Java 会分配一个包含 64 位的 box，无论对象类型如何。
64 位的box中不包含类型内容的数据，而是包含类在内存中的地址

```java
Walrus someWalrus;
/** - Java 在内存中开辟了一个名为 someWalrus 的“盒子”。这个盒子的大小固定为 64位 (64 bits)。 此时盒子里还没有存有效信息（或者是 null，即全 0）。*/

someWalrus = new Walrus(1000,8.3);
/** new 这个关键字命令 Java 在内存的另一个区域（堆内存，Heap）真正“造”出一只海象 */
/** 这只真海象在内存中有一个起始编号（真正的内存地址） */

/** = */
/** 使用了GRoE法则 */
/** new 操作符返回了那只海象的地址（那串 64 位的二进制数）。等号把这串“地址位”复制到了 someWalrus 这个盒子里。 */
```
