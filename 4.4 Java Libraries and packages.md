# `ADT`（抽象数据类型）
> 数据结构的核心依赖于抽象思想与机械执行的分离,分别对应**抽象的接口**与**具体的实现**

接口在 Java 中除了定义数据结构外，还有两个主要功能：
- **规范实现规格与能力**：接口提供一个正式契约，保证特定行为。
- **函数包裹**：由于 Java 不允许直接传递函数作为参数，接口用于将函数包裹在对象内


**使用接口继承允许开发者定义对象能做什么，有效地将用户的抽象宇宙与实现管理的内部机制“隔离”起来。**

## 实现不同数据结构底层实现的思考


- 对于 栈 这类操作受限的结构，链表和数组均可胜任，链表在规避扩容开销上略具优势。
    
- 对于需要 随机访问 的结构（如抓取袋），数组由于其天然的索引特性，在性能上具有绝对优势。

**在实现某个数据结构时，学会思考使用哪个基础数据结构最优**
